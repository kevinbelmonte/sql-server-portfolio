# SQL Server – Junior Portfolio

Proyecto simple para demostrar conocimientos básicos de SQL Server.

## Contenido
- Creación de tablas
- Inserción de datos
- SELECT básicos
- JOIN
- GROUP BY y SUM

## Cómo ejecutar
1. Ejecutar `database/01_create_tables.sql`
2. Ejecutar `data/02_insert_data.sql`
3. Ejecutar consultas en la carpeta `queries`

## Nivel
Junior
